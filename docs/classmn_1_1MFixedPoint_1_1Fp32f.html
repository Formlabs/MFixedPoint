Detailed Description

template<uint8_t q>
class mn::MFixedPoint::Fp32f< q >

The template argument p in all of the following functions refers to the fixed point precision (e.g. q = 8 gives 24.8 fixed point functions). Contains mathematical operator overloading. Doesn't have modulus (%) overloading
